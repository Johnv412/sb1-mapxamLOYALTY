A comprehensive loyalty rewards system for WordPress.

== Description ==

LoyaltyEngine is a powerful loyalty rewards plugin for WordPress that allows you to create and manage a points-based loyalty program for your customers.

Features:
* Customer point tracking
* Reward redemption system
* Gift card management
* Admin dashboard for managing rewards and points
* Shortcodes for displaying user points and available rewards
* Gutenberg blocks for easy integration

== Installation ==

1. Upload the `loyalty-engine` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure the plugin settings in the LoyaltyEngine menu

== Frequently Asked Questions ==

= How do customers earn points? =

Points can be awarded based on purchases, actions, or manually by the admin.

= Can I customize the reward options? =

Yes, you can create and manage custom rewards through the admin dashboard.
